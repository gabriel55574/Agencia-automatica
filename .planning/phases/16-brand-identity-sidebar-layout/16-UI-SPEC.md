## Color

Hybrid theme per D-02: sidebar uses Dark theme tokens, content area uses Light theme tokens.

### Sidebar (Dark Theme)

| Role | Value | Usage |
|------|-------|-------|
| Background | `#062222` (Rich Black) | Sidebar 240px panel |
| Text (default) | `#F2F2F2` (Smoke White) | Nav item labels, sign-out text |
| Text (secondary) | `#8A9999` (Gray 300) | Disabled or muted items if needed |
| Accent | `#BFF205` (Electric Lime) | Active item indicator bar, active item text |
| Active background | `rgba(191, 242, 5, 0.08)` | Subtle lime tint behind active item |
| Hover background | `rgba(242, 242, 242, 0.06)` | Subtle highlight on hover |
| Border (divider) | `rgba(242, 242, 242, 0.1)` | Separator above sign-out area |

### Content Area (Light Theme)

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#F2F2F2` (Smoke White) | Page background |
| Secondary (30%) | `#FFFFFF` (White) | Cards, surfaces |
| Accent (10%) | `#BFF205` (Electric Lime) | Reserved for: primary CTA buttons, active state indicators, focus rings |
| Text primary | `#062222` (Rich Black) | Headings, body text |
| Text secondary | `#5C6E6E` (Gray 400) | Descriptions, timestamps |
| Text accent | `#6D8A03` (Lime Dark) | Accent text on light backgrounds — WCAG safe |
| Destructive | `#EF4444` (Red 500) | Delete/archive actions only |
| Border default | `#E8EDED` (Gray 100) | Card borders, dividers |

**WCAG constraint (D-05):** Electric Lime `#BFF205` must NEVER be used as text on light backgrounds (1.4:1 contrast ratio = FAIL). Use Lime Dark `#6D8A03` instead. Electric Lime on Rich Black `#062222` = 12.1:1 (AAA PASS).

Accent reserved for: active sidebar indicator, primary CTA buttons, focus outline rings, badge highlights. Never for body text on light surfaces.

## Sidebar Visual Specification

### Layout Structure

```
+--[Sidebar 240px]--+--[Content Area flex-1]------+
|                    |                              |
| [Logo area]        |  <main> page content         |
| --space-6 padding--|                              |
|                    |                              |
| [Dashboard]        |                              |
| [Clientes]         |                              |
| [Custos]           |                              |
| [Analytics]        |                              |
| [Templates]        |                              |
|                    |                              |
| ...spacer...       |                              |
|                    |                              |
| [border-top]       |                              |
| [Sair]             |                              |
+--------------------+------------------------------+
```

### Sidebar Item Anatomy

```
+--[3px lime bar]--[16px pad]--[icon 24px]--[8px gap]--[label]--[16px pad]--+
|                                                                             |
| Height: 44px (touch target)                                                |
| Icon: Lucide, 24px, currentColor                                          |
| Label: Roboto 14px/400, Smoke White                                       |
| Active: lime bar visible + text #BFF205 + bg rgba(191,242,5,0.08)         |
| Hover: bg rgba(242,242,242,0.06)                                          |
+-----------------------------------------------------------------------------+
```

### Active State (D-13)

- Left edge: 3px vertical bar, `#BFF205` (Electric Lime), full height of item
- Text color: `#BFF205` (Electric Lime)
- Background: `rgba(191, 242, 5, 0.08)` (Lime ~8% opacity)
- Icon color: `#BFF205` (inherits from text via currentColor)

### Icon Mapping (D-14, D-15)

| Nav Item | Lucide Icon | Import Name |
|----------|-------------|-------------|
| Dashboard | LayoutDashboard | `LayoutDashboard` |
| Clientes | Users | `Users` |
| Custos | DollarSign | `DollarSign` |
| Analytics | BarChart3 | `BarChart3` |
| Templates | FileText | `FileText` |
| Sair | LogOut | `LogOut` |

### Mobile Behavior (D-12)

- Breakpoint: 1024px (`lg:` in Tailwind)
- Below 1024px: sidebar hidden, slim top bar (48px height) with:
  - Left: hamburger icon (Menu from Lucide, 24px)
  - Center: Velocity logo (icon-only variant, `Logo/logo tipo fundo claro (icone).svg`)
  - Right: empty (or minimal)
- Hamburger tap: sidebar slides in as overlay from left
  - Overlay backdrop: `rgba(4, 25, 25, 0.8)` (Black Deep 80%)
  - Transition: 250ms ease (--transition-normal)
  - Close: tap backdrop or X button or nav item selection

## CSS Custom Properties Contract

All tokens from `docs/VELOCITY-UI-IDENTITY-SYSTEM.md` section 4 must be injected into `src/app/globals.css`. The full `:root` block includes:

- 3 primary colors (black, lime, white)
- 8 derived colors (lime-hover, lime-active, lime-light, lime-dark, black-soft, black-deep)
- 4 grays (100-400)
- 4 semantic colors (success, error, warning, info)
- 2 font families (heading, body)
- 11 typography shorthand tokens
- 12 spacing tokens (space-1 through space-24)
- 6 border radius tokens
- 4 shadow tokens
- 3 transition tokens

Light and dark theme tokens from section 5 must be declared as:
- `:root` (defaults to light theme for content area)
- `.sidebar, [data-theme="dark"]` (dark theme for sidebar)

## Focus States

Per identity doc section 10.2:
- Outline: 2px solid `#BFF205` (Electric Lime), offset 2px
- All interactive sidebar items must have visible focus ring
- Tab order: logo area (skip link target) -> Dashboard -> Clientes -> Custos -> Analytics -> Templates -> Sair
- `prefers-reduced-motion`: disable sidebar slide animation, use instant show/hide
