<objective>
Create a reusable Breadcrumb component and add hierarchical breadcrumb navigation to all client detail pages, so the operator always knows where they are in the page hierarchy and can navigate back to any ancestor page with a single click.
</objective>

<threat_model>
## Threat Model

**Attack surface:** Breadcrumb labels come from database fields (client name). Links are internal Next.js routes constructed from URL params (client ID).

| ID | Threat | Severity | Mitigation |
|----|--------|----------|------------|
| T-17-02 | XSS via client name in breadcrumb | Low | React auto-escapes text content rendered via JSX `{variable}`. Client names are rendered as text nodes, never `dangerouslySetInnerHTML`. No risk. |
| T-17-03 | Open redirect via breadcrumb href | Low | All hrefs are hardcoded internal paths (`/clients`, `/clients/{id}`). No user-controlled URL construction. |

**Residual risk:** None. Standard React text rendering with internal links.
</threat_model>

```tsx
import Link from 'next/link'
import { ChevronRight } from 'lucide-react'

export interface BreadcrumbItem {
  label: string
  href?: string
}

interface BreadcrumbProps {
  items: BreadcrumbItem[]
}

/**
 * Hierarchical breadcrumb navigation for detail pages.
 *
 * Usage:
 *   <Breadcrumb items={[
 *     { label: 'Clientes', href: '/clients' },
 *     { label: 'Joao Silva', href: '/clients/abc123' },
 *     { label: 'Editar' },
 *   ]} />
 *
 * Last item has no href (current page).
 * D-08: Chevron separator (ChevronRight, 14px)
 * D-09: Ancestors in gray-300 with hover lime-dark; current page in text-primary font-medium
 * D-10: Only used on detail pages, never on root pages
 * D-13: Positioned above page title with mb-2 (8px)
 */
export function Breadcrumb({ items }: BreadcrumbProps) {
  if (items.length === 0) return null

  return (
    <nav aria-label="Breadcrumb" className="mb-2">
      <ol className="flex items-center gap-1 text-sm">
        {items.map((item, index) => {
          const isLast = index === items.length - 1

          return (
            <li key={index} className="flex items-center gap-1">
              {index > 0 && (
                <ChevronRight className="h-3.5 w-3.5 text-zinc-400 shrink-0" />
              )}
              {isLast || !item.href ? (
                <span className="text-zinc-900 font-medium">{item.label}</span>
              ) : (
                <Link
                  href={item.href}
                  className="text-zinc-400 hover:text-[#6D8A03] transition-colors"
                >
                  {item.label}
                </Link>
              )}
            </li>
          )
        })}
      </ol>
    </nav>
  )
}
```

Key details:
- `ChevronRight` from lucide-react at `h-3.5 w-3.5` (14px per D-08)
- Ancestor links: `text-zinc-400` (maps to --color-gray-300 intent per D-09) with `hover:text-[#6D8A03]` (Lime Dark per accessibility requirement from Phase 16)
- Current page (last item): `text-zinc-900 font-medium` (D-09)
- `aria-label="Breadcrumb"` for accessibility
- `mb-2` on the nav element for 8px margin below (D-13)
- Renders as `<nav><ol><li>` for semantic HTML
