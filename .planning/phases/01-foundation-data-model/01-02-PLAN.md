       RETURNS SETOF squad_jobs AS $$
       BEGIN
         RETURN QUERY
         UPDATE squad_jobs
         SET status = 'running', started_at = NOW()
         WHERE id = (
           SELECT id FROM squad_jobs
           WHERE status = 'queued'

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Application -> Database | All writes go through Supabase client; RLS validates auth |
| Direct SQL -> Database | Triggers enforce business rules regardless of access path |
| Anon key -> Tables | RLS prevents unauthenticated access to all tables |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-01 | Tampering | phases table | mitigate | `enforce_phase_sequence` trigger prevents skipping phases at DB level, not just app code |
| T-02-02 | Information Disclosure | all tables | mitigate | RLS enabled on all 6 tables; only `authenticated` role has access |
